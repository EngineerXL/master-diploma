# Выпускная квалификационная работа студента СпецВО-магистратуры

## Разработка алгоритма лидарной одометрии беспилотного автомобиля, учитывающего перекрытие поля зрения

Студент: Инютин Максим Андреевич

Группа: М8О-206СВ-24

Научный руководитель: Ухов П. А.

## Цель и задачи работы

**Цель:** разработать устойчивый алгоритм лидарной одометрии, сохраняющий оценку движения на коротком участке пути.

**Объект:** локализация беспилотного автомобиля по лидарным данным.

**Предмет:** алгоритмы лидарной одометрии при перекрытии поля зрения.

**Задачи:**

1. Анализ классических методов регистрации облаков точек;
2. Формализация сценария перекрытия поля зрения;
3. Разработка алгоритма: вокселизация, взвешенный ICP, модель движения, фильтр Калмана;
4. Реализация конвейера и метрик качества;
5. Экспериментальное исследование на Boreas-RT.

## Математическая постановка задачи

По паре соседних облаков $(P_{t-1}, P_t)$ оценить относительное преобразование $T_t$ и скорости $(v_x,v_y,v_z,\omega_x,\omega_y,\omega_z)$.

**Образ результата:**

- Устойчивость к перекрытию поля зрения.
- Корректная оценка на 10 секундах ($\sim$ 100 кадров).
- Подавление выбросов и учёт кинематики.
